CSV Format

swimmer_name,coach_name,region,stroke,distance,time Amina Ochieng,Coach Mutua,Nairobi,Freestyle,100m,58.40 Brian Kipchoge,Coach Waweru,Eldoret,Butterfly,100m,1:05.20 Cynthia Auma,Coach Otieno,Kisumu,Backstroke,200m,2:35.00
  • swimmer_name — full name
  • coach_name — coach's name (auto-created if new)
  • region — Nairobi, Mombasa, Kisumu, Nakuru, Eldoret, Nyeri
  • stroke — Freestyle, Backstroke, Breaststroke, Butterfly, IM
  • distance — 50m, 100m, 200m, 400m, 800m, 1500m
  • time — seconds or MM:SS.ss
